# BizTextMaker

## 概要
BizTextMaker は、
ビジネス文書を簡単に作成し、プレビュー・印刷・PDF保存できることを目指したWebアプリです。

※アプリ開発初学者の為、いたってシンプルな構造となっています。

---
## 制作目的
実務経験をもとに、文書作成の際、ExcelやWordを複数使用し手順が複雑かつ、編集や同期させる必要があり、他の人が使用する際に手順書が必要でした。

そのため、直感的に使用しやすいシンプルな設計を目指しています。複雑な工程はふまず印刷まで簡潔できるアプリ作り、簡単な引継ぎで済むことを目的としています。


## 反省点
自分だけでは改善案が最適ではなく、AIの手伝いをもらいました。
お叱りもたくさん受けました。具体的には

・無駄なコードやcssでの指定が原因で衝突が起こる
・一度したものが動かなくなる
・コード自体の知識不足
・コード記述の際の優先順位

など、もっと理解を深め、大枠からどのように分岐させて、ファイルを構成やコードの作成をさせていくかが今後の課題点です。


## 主な機能

* 日付指定
* 文書の種類より文書テンプレート切替
* 得意先入力・編集・削除（改良予定）
* 備考欄入力
* 文書プレビュー表示
* 印刷 / PDF保存
