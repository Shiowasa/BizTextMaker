# BizTextMaker

## 概要

BizTextMaker は、
ビジネス文書を簡単に作成し、プレビュー・印刷・PDF保存できる Webアプリです。

JavaScript 初学習として制作しました。

---

## 主な機能

* 文書テンプレート切替
* 得意先入力
* 備考欄入力
* 文書プレビュー表示
* 印刷 / PDF保存
* localStorage を利用したデータ受け渡し
* 印刷時ボタン非表示

---

## 使用技術

* HTML
* CSS
* JavaScript

## 工夫した点

### 改行保持

textarea の改行を保持するため、
replace() を使用して `<br>` へ変換。

```javascript
documentContent.replace(/\n/g, "<br>");
```

---

### 画面間データ受け渡し

localStorage を利用し、
index.html → preview.html へ文書データを受け渡し。

---

### 印刷専用CSS

印刷時のみボタンを非表示化。

```css
@media print {

    .no-print {
        display: none;
    }

}
```
## 実務を想定し、不要部分の非表示・位置の調整
・非表示で良い物
    "<h2>"→コメントアウトで対応
・実際の印刷を想定し、どこに寄せるか
    その中でも自社情報などブロックは右寄せがいいが、ブロック内では左寄せが良いなど

## 反省点
自分だけでは改善案が最適ではなく、AIの手伝いをもらいました。
お叱りもたくさん受けました。具体的には

・無駄な部分やcssでの指定が原因で衝突が起こる
・一度実働したものが動かなくなる
・コード自体の知識が不足しておりどのように構築で迷走
・コード記述の際の優先順位

など、もっと理解を深め、大枠からどのように分岐させて、ファイルを構成やコードの作成をさせていくかが今後の課題点です。
